# Заглушки для реализации

Эти файлы созданы как заглушки для будущих реализаций:

- `src/export_config.cpp` - реализация класса ExportConfig (QObject обертка)
- `src/specifier_config.cpp` - реализация класса SpecifierConfig (QObject обертка)

Текущая реализация использует структуры из `types.h`, которые полностью функциональны.
При необходимости можно расширить классы-обертки для лучшей интеграции с Qt сигналами/слотами.
